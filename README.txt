== Description ==

**The Terra Themes Tools plugin is meant to be used only with themes from Terra Themes. Other themes might not work with the extended features.**

Terra Themes Tools registers custom post types (like Projects, Employees and more) with custom fields for themes from Terra Themes.

== Installation ==

1. Go to Plugins > Add New and search for Terra Themes Tools. Install and activate it.
2. That's it. No configuration needed.
